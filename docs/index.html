Scan Modes

Single target, CIDR ranges, or credential files. Maelstrom adapts its execution strategy to match your engagement scope.

Single Target

Full enumeration against one host. Anonymous probing auto-tests null and guest sessions. Credential mode unlocks all 35+ modules with parallel execution.

maelstrom 10.0.0.1 -u admin -p pass -d CORP

Multi-Target

CIDR ranges, IP ranges, and target files. Parallel host discovery with port 445 pre-scan. Per-target results with aggregate summary.

maelstrom 10.0.0.0/24 -u admin -p pass

Multi-Credential

Test multiple credentials from a file. Visual share access matrix, admin detection, and per-user access comparison across all discovered shares.

maelstrom 10.0.0.1 -C creds.txt -d CORP

Features

Reconnaissance-only enumeration powered by NetExec. Every module reads data without executing commands on targets.

35+ Enum Modules

Users, groups, shares, LAPS, Kerberoastable, delegation, ADCS, gMSA, GPP, PSO, SCCM, and more. Run all with -A or select specific modules with individual flags.

Multi-Protocol

SMB, LDAP, MSSQL, RDP, FTP, NFS, and VNC enumeration in a single scan. Port availability is auto-detected before each module executes.

Actionable Next Steps

Findings generate ready-to-run commands with credentials auto-filled. Priority-ranked with auto-exploit warnings for dangerous tools like secretsdump.

Parallel Execution

ThreadPoolExecutor with 15 workers for module-level parallelism, 5 workers for multi-target, and 100 for port pre-scanning. Proxy mode auto-reduces to 2.

Copy-Paste Output

Clean line-by-line lists of usernames, SPNs, shares, computers, and more. Pipe directly to other tools or save to files for later processing.

Credential Flexibility

Password, NTLM hash, Kerberos (ccache/AES), and certificate-based PKINIT auth. Anonymous mode auto-probes null and guest sessions.

Catppuccin Terminal UI

24-bit true color output with Catppuccin Mocha palette. enum4linux-ng style indicators: [*] info, [+] success, [-] error, [!] warning.

Proxy Mode

Full proxychains and SOCKS support. Auto-reduces concurrency, increases timeouts, and skips incompatible modules. Detected via LD_PRELOAD.

Zero Dependencies

Pure Python 3.10+ with only stdlib imports. No pip packages, no virtual environments, no build steps. Just Python and NetExec on your PATH.

Architecture

Seven-phase execution pipeline. Pre-scan, validate, cache, enumerate in parallel, then generate reports with next steps.

1

Pre-scan & Validation

Port 445 TCP scan filters live hosts from CIDR ranges. SMB reachability and hostname resolution verify each target before enumeration begins.

2

Auth & Cache Prime

Anonymous probing tests null/guest/LDAP sessions. Credential validation then cache priming with 7 parallel queries: SMB, RID brute, LDAP, pass policy.

3

Parallel Enumeration

Sequential domain intel pass followed by 36 independent modules via ThreadPoolExecutor with 15 workers. Proxy mode reduces to 2 workers.

4

Reporting

Executive summary, actionable next steps with auto-filled credentials, share access matrix for multi-cred mode, and copy-paste lists for tool chaining.

Getting Started

Install with pipx, point at a target, and let Maelstrom handle the rest. No configuration needed.

install.sh 
# Install globally with pipx
$ pipx install git+https://github.com/Real-Fruit-Snacks/Maelstrom.git

# Full enumeration with credentials
$ maelstrom 10.0.0.1 -u admin -p 'Password123' -d CORP

# Anonymous enumeration
$ maelstrom 10.0.0.1

# Pass-the-hash
$ maelstrom 10.0.0.1 -u admin -H aad3b435:31d6cfe0
advanced.sh 
# Multi-target CIDR scan
$ maelstrom 10.0.0.0/24 -u admin -p pass

# Multi-credential comparison
$ maelstrom 10.0.0.1 -C creds.txt -d CORP

# Specific modules only
$ maelstrom 10.0.0.1 -u admin -p pass --shares --laps --adcs

# Discover live SMB hosts
$ maelstrom 10.0.0.0/24 --discover-only

Detection Awareness

Maelstrom uses standard enumeration protocols. Understand what generates alerts and what blends in.

Low Visibility

  • LDAP queries (standard directory lookups)
  • SMB share enumeration (normal file server access)
  • DNS enumeration recommendations (passive only)
  • Anonymous/guest session probing
  • RPC enumeration (standard Windows calls)

May Generate Alerts

  • Port 445 pre-scan across CIDR ranges
  • RID brute-force enumeration
  • Multiple protocol connections in rapid succession
  • Multi-credential testing (potential spray detection)
  • Large-scale share spider operations
